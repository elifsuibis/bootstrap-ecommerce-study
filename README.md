# Mart Teknoloji - E-Ticaret Arayüz Çalışması
Bu proje,  **front-end** pratiği yapmak için **Bootstrap 5** kullanılarak geliştirilmiş bir teknoloji mağazası ana sayfa tasarımıdır. 

##  Kurulum
1. İlgili klasöre projeyi klonlayın.
2. Klasör içindeki `index.html` dosyasını bir tarayıcı ile açın.

## İçerik Bilgisi
Geliştirilen sayfadaki 'Mart Teknoloji' markası ve ürünler tamamen hayalidir. Gerçek kurum ya da ürünlerle bağlantılı değildir.Kullanılan görseller yapay zeka aracı kullanılarak hazırlanmıştır.
